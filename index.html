Шкаф управления автоматизированным тепловым пунктом

ШУАТП предназначен для управления индивидуальным тепловым пунктом в автоматическом и ручном режимах.

Спроектирован и собран на базе микропроцессорного регулятора АРТ-05.

Диспетчеризация - оперативный контроль и управление технологическими процессами.

Описание

ШУАТП в зависимости от назначения может управлять:
  • done насосами отопления, ГВС, вентиляции и подпитки;
  • done регулирующими клапанами с электроприводами систем отопления, ГВС, вентиляции и т.д.
  • done запорными клапанами системы подпитки отопления.
ШУАТП обеспечивает:
  • done автоматическое регулирование температуры системе отопления по наружному воздуху;
  • done поддержание заданной температуры горячей воды;
  • done суточное отклонение температурного графика (например: снижение заданной температуры ГВС в ночные часы);
  • done автоматическое переключение насосов - основной\резервный;
  • done включение резервного насоса в случае аварии основного;
  • done автоматическую подпитку систем отопления/вентиляции по сигналу реле давления;
  • done дистанционное управление и наблюдение за параметрами;
ШУАТП обеспечивает полную защиту насосного оборудования:
  • done от короткого замыкания;
  • done от «сухого хода»;
  • done от перегрузки по току;
  • done от перегрева обмоток электродвигателя;
  • done от обрыва фаз и некачественного электропитания;
